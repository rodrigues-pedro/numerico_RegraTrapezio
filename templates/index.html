Sintaxe para função:

Para $x\cdot a$ escreva x*a

Para $\frac{a}{b}$ escreva x/a

Para $x^a$ escreva x**a

Para sen(x) escreva sin(x)

Para $\cos(x)$ escreva cos(x)

Para $\ln(x)$ escreba log(x)

Para $\log_a x$ escreva log(x, a)

Para $e^x$ escreva exp(x)
